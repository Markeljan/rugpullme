## RugPullMe

### Install

```shell
$ forge install OpenZeppelin/openzeppelin-contracts --no-git
```

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Deploy

```shell
$ forge script script/Counter.s.sol:CounterScript --rpc-url <your_rpc_url> --private-key <your_private_key>
```
